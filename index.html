Projects

AHAbot

Working on a team of four over a three-week period, we created a slackbot with the ability to control a user's Home Assistant on a Raspberry Pi. After Home Assistant device registration, users are able to chat with the bot in a Slack workspace, manipulate smart devices in their homes, and get real-time status updates for those smart devices. The bot was built in Node.js using Node Slack SDK, Natural.js, and axios. The registration site is built in Django.

Question Box

An application built to mimic the functionality of Stack Overflow. Registered users can ask questions and post answers. Any user can view questions and answers to questions. Question authors can mark answers as correct. In this project, I and a team of two other classmates use AJAX to update the page when answers are posted to prevent a page refresh. The site is built in Django.

iTunes Music Search Application

This application gives a user the ability to search the iTunes library via the iTunes API and return song, album, and artist information. Search terms can include song titles, artist names, or album titles. The user can listen to any song preview that the iTunes library provides for free. The app is built primarily using vanilla JavaScript.
